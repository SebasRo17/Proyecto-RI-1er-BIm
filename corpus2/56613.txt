Consider the following `ImplicitRegion`:               reg = ImplicitRegion[x^2 + y^2 + z^2 == 16, {x, y, z}];      We can discretize it using `DiscretizeRegion`               dr = DiscretizeRegion[reg]      ![Mathematica graphics](http://i.stack.imgur.com/g0hE2.png) To get finer triangles the option `MaxCellMeasure` is available, only it does nothing when used.               DiscretizeRegion[reg, MaxCellMeasure -> {"Area" -> #}] & /@ {0.1, 0.05}      ![Mathematica graphics](http://i.stack.imgur.com/iEGOh.png) Of course using one of the `Graphics` primitives               DiscretizeRegion[Ball[], MaxCellMeasure -> {"Area" -> #}] & /@ {0.1, 0.05}      ![Mathematica graphics](http://i.stack.imgur.com/DO7W2.png) You can see that it works fine.
