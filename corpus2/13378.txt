I'm trying to plot $a x^2 + b x + c$ where $a, b, c$ (and $x$) take on a range of values. I tried               ParametricPlot[{x, a x^2 + b x + c}, {x, -1, 1},                    {a, -.25, .25}, {b, -.25, .25}, {c, -.25, .25}]      but `ParametricPlot` can only handle up to two parameters. My work around uses `FindInstance` which can be really slow:               RangePlot[eq_Equal, {x_, xmin_, xmax_}, {y_, ymin_, ymax_}, vars_List, constraints_] :=          RegionPlot[             FindInstance[And[constraints, eq], vars] =!= {}, {x, xmin, xmax}, {y, ymin, ymax}         ]          RangePlot[y == a x^2 + b x + c, {x, -1, 1}, {y, -1, 1}, {a, b, c},          -.25 < a < .25 && -.25 < b < .25 && -.25 < c < .25]      ![result of RangePlot\[\]](http://i.stack.imgur.com/62yng.png) Is there a better way to do this?
