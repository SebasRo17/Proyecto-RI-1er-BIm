What is difference between `protected static` method and `protected` method? For example `demo` and `demo2` in `Data2` respectively Data.java:               package pack1;          import pack2.*;          public class Data extends Data2 {              public static void main(String args[]){             System.out.println("From main class");             Data d=new Data();             d.first();          }              void first(){             Data2 d2=new Data2();                  d2.demo();             d2.demo2();         }     }      Data2.java:               package pack2;          public class Data2 {         protected static void demo(){             System.out.println("In method demo");         }              protected void demo2(){             System.out.println("In method demo2");         }     }
