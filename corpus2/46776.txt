I'm matching a pattern with `Repeated (..)` How can I use the repeat count in the replacement rule? Example:                Cases[ { {0, 2, 2, 5, 2, 4, 1, 2}, {0, 1, 2, 3, 4, 5}} ,          {0, Repeated[x_ /; x != 5], 5, __} :> {0, Sequence @@ Table[x, {2}], 5}]                                                                          ^ <---?      > {{0, 2, 2, 5}} Obviously instead of hard coding the `2` I want the length of the repeat.. My workaround for this example is this:                #[[;; First@First@Position[#, 5]]] & /@          Cases[ { {0, 2, 2, 5, 2, 4, 1, 2}, {0, 5, 5, 1, 2, 3, 4, 5}} ,            {0, Repeated[x_ /; x != 5], 5, __} ]      > {{0, 2, 2, 5}} It seems there should be a cleaner solution though.
