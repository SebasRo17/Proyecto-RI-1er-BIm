The documentation for `RevolutionPlot3D` shows that you can specify a parametric curve like this:               RevolutionPlot3D[{u, u}, {u, 0, 1}]      ![Mathematica graphics](http://i.stack.imgur.com/Zppvk.png) I'd like to do this with a `Piecewise` function for `{fx,fz}`:               RevolutionPlot3D[Piecewise[{{{u, u}, u < 0.5}, {{u, 0.5}, u >= 0.5}}], {u, 0,        1}]      but I get messages and a blank plot: > Dot::rect: Nonrectangular tensor encountered. ![Mathematica graphics](http://i.stack.imgur.com/o57ob.png) whereas a `Piecewise` function for `{fx,fz}` works in `ParametricPlot`, for example:               ParametricPlot[Piecewise[{{{u, u}, u < 0.5}, {{u, 0.5}, u >= 0.5}}], {u, 0, 1}]      ![Mathematica graphics](http://i.stack.imgur.com/1pJqp.png) By writing `Piecewise` expressions for `fx` and `fz` separately, it works:               RevolutionPlot3D[{Piecewise[{{u, u < 0.5}, {u, u >= 0.5}}],          Piecewise[{{u, u < 0.5}, {0.5, u >= 0.5}}]}, {u, 0, 1}]      ![Mathematica graphics](http://i.stack.imgur.com/LtPbf.png) How is it possible to piece together a function for `{fx,fz}` in `RevolutionPlot3D`?
