Which is better: many class definitions in same file or every class definition in separate file? **Examples:** 1) Many classes in same file. Say, `myManyClasses.cs`:               namespace myPack     {        class myClass1()        {        }        class myClass2()        {        }        class myClass3()        {        }       .        .        .     }      2) Every class in separate file: _myClass1.cs_               namespace myPack     {        class myClass1()        {        }     }      _myClass2.cs_                namespace myPack      {        class myClass2()        {        }       }      **. . .**
